## 📖 项目概述

MT-Launcher 是一款专为 Android 电纸书（Eink设备）优化的轻量级启动器，具有以下核心特性：

- 极简交互设计，降低墨水屏刷新频率
- 深度适配电子墨水屏显示特性
- 支持应用分类管理和快捷操作
