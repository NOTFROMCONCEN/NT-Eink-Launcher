# MT-Launcher 梅糖桌面

## 项目概述
MT-Launcher 梅糖桌面是一款专为搭载 Android 系统的 Eink 电纸书设计的启动器，旨在为用户提供简洁且实用的桌面体验。

## 跨平台说明
本项目原计划通过 `Kotlin` 混合编程实现跨平台，理论上可支持 Mac、Linux、Windows 等平台。但由于许多 API 需要重写才能实现相应功能，目前暂时放弃跨平台开发，日后有时间会再进行维护。
